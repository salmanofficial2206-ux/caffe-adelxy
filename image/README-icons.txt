# Ikon PWA untuk Coffe Adel

Untuk membuat ikon PWA yang sempurna, Anda perlu:

1. **Ikon 192x192 pixels** (icon-192.png)
2. **Ikon 512x512 pixels** (icon-512.png)

## Cara membuat ikon:

1. Buka file `icon.svg` yang sudah dibuat
2. Gunakan tools online seperti:
   - https://cloudconvert.com/svg-to-png
   - https://convertio.co/svg-png/
   - Atau software seperti Inkscape/Photoshop

3. Konversi SVG ke PNG dengan ukuran:
   - 192x192 pixels untuk icon-192.png
   - 512x512 pixels untuk icon-512.png

4. Simpan file di folder `image/` dengan nama:
   - `icon-192.png`
   - `icon-512.png`

## Placeholder files (hapus setelah membuat ikon PNG):

File ini adalah placeholder sementara. Ganti dengan file PNG asli setelah konversi dari SVG.
